Stylized Knight

A personal project based on Thomas Karlsson's Slime Armor Character Concept. I wanted to capture Thomas' toon style in a modern game optimized character. Modeled in Maya and ZBrush. Textured in Substance Painter. Rendered In Unreal Engine 4. 22.3k triangles.
